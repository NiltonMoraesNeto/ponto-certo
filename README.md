# React + TypeScript + Vite

Projeto FrontEnd

Projeto feito em [React](https://react.dev/) com [Vite](https://vite.dev/), [TypeScript](https://www.typescriptlang.org/), estilização com [TailwindCSS](https://tailwindcss.com/) e [material/ui](https://mui.com/material-ui/) e o para a tradução das páginas é usado o [i18next](https://www.i18next.com/overview/getting-started). O backEnd é usando o [JsonServer](https://github.com/typicode/json-server) para simular as apis.
## Getting Started

```bash
npm i
npm run dev
npm run json-server       
```

Abrir [http://localhost:5173/login](http://localhost:5173/login) para a tela de login
